Aanya is a powerful, locally-installed AI assistant designed to bridge the gap between natural language and system-level control. Built with Python and powered by the ultra-fast Groq LPU, Aanya allows users to control their PC, manage files, and execute complex workflows using simple voice commands.

Key Features:

🎙️ Voice-Native Control: Real-time speech recognition and natural sounding TTS.

⚡ "God Mode" Execution: Dynamic Python script generation for complex, non-standard tasks.

🖥️ System Integration: Control volume, brightness, power states, and application launching.

🛡️ Privacy-First Architecture: Local processing for sensitive data; minimal cloud dependence.
